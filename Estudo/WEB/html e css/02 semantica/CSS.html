2.1 - Entendendo CSS

O css pode ser em modo inline(metodo não mais utilizado) e deve ser abandonado, pois gera um codigo confuso e saturado. Existe o metodo interno utilizando a tag style no head do html é uma forma mais facil, mas ainda não ideal varias desvantagens, entre elas que o estilo daria um codigo muito maior que a pagina, e se houvesse 30 paginas, teria que copiar em todas as paginas.
A forma correta é utilizar um arquivo externo .css, importando para o arquivo interno atraves da tag link, em realidade em todo desenvolvimento é comum haver a separação de arquivos.
