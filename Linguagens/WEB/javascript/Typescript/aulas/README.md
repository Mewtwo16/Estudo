# Aulas Folder Documentation

This folder contains TypeScript code and resources for educational purposes. The main entry point is located in the `src/index.ts` file.

## Purpose

The "aulas" folder is designed to provide a structured environment for learning and experimenting with TypeScript. It includes examples and exercises that can be executed independently.

## Getting Started

To get started with the "aulas" folder, follow these steps:

1. **Install Dependencies**: Navigate to the `aulas` directory and run the following command to install the necessary packages:

   ```
   npm install
   ```

2. **Build the Project**: To compile the TypeScript code, use the following command:

   ```
   npm run build
   ```

3. **Run the Code**: After building, you can run the code using:

   ```
   npm start
   ```

## Structure

- `src/index.ts`: The main entry point for the TypeScript code.
- `package.json`: Contains npm configuration and scripts.
- `tsconfig.json`: TypeScript configuration file.

Feel free to explore and modify the code in this folder to enhance your learning experience!
