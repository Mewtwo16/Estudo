1.2 - Tags semanticas

As tags semanticas são as tags que servem para tornar o HTML5 semantico e ajudam o navegador a encontrar mais logica no seu site.
Há formas semanticas e formas não semanticas de utilizar as tags, inclusive a possibilidade de combinar tags e gerar um texto de facil comrpeenção.
A tag B apenas da forma em negrito(Sem semantica). Enquanto a tag strong cria um texto em destaque.
A tag I define um texto em italico(sem semantica). Enquanto a tag em define um texto em enfases.
A tag mark cria um texto marcado como um marca texto.
Exemplo = Texte de mark
A tag small cria um texto pequeno.
A tag del cria um texto excluido(algo que deve ser lido não deve ser considerado).
Exemplo = Este texto foi deletado
A tag ins cria um texto inserido
Exemplo = Este texto foi incluido
As tags sub e sup servem para deixar textos sub escritos ou superescritos
Exemplo = Este é um texto com subescrito. Enquanto este é um texto com um supescrito. A tag q cria uma citação.
Exemplo = Agua mole em pedra dura, tanto bate até que fura!
